## 企业库存管理系统 inventory-app

完成了一个完整的企业库存管理 Web 应用，项目路径：`e:/code/workbuddy/System/inventory-app`

### 技术栈
- React + TypeScript + Vite + Tailwind CSS v4 + Recharts + Lucide React

### 功能模块
- **登录/认证系统**：两种角色（销售/经理）
- **工作台 Dashboard**：角色差异化数据展示、待办事项、库存预警
- **新建取货申请（销售）**：选产品/数量/价格/客户/项目，支持草稿保存和提交审核
- **我的申请（销售）**：查看订单状态、处理经理修改意见、确认完成销售
- **审核中心（经理）**：批准/拒绝/调整数量价格，附带意见
- **全部订单（经理）**：多条件筛选搜索，展开查看详情
- **货仓管理（经理）**：库存数量、出入库记录、手动库存调整
- **统计报表**：月度营收趋势折线图、产品排行柱状图、销售人员饼图占比、月度汇总表

### 关键文件
- `src/types/index.ts` - TypeScript 类型定义
- `src/data/mockData.ts` - 模拟初始数据
- `src/store/AppContext.tsx` - 全局状态管理（useReducer）
- `src/components/*.tsx` - 各功能页面组件
